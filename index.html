MD/hv-config builds under test

  • OpenSPARC_T1_original — Sun's unmodified 2006 release (no fix applied; expected to fail on modern kernels per Experiment #16, not yet confirmed per-image)
  • OpenSPARC_T1_rebuild — same sources, rebuilt with our mdgen; byte-identical to OpenSPARC_T1_original, so same expected boot behavior
  • snv136/256, snv136/1024, snv136/2048 — memory/cpus/cache + mmu-* fix applied (see patches/snv_136/), 256/1024/2048 MiB variants
  • 1GiB-snv_77 (1GiB-experimental) — externally-sourced prebuilt blobs, not built from src/
